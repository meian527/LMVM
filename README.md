# Lamina-VM 

## 目录
- [特性]
- [快速开始]
  ```bat
  cmake
  .\lmvm.exe
  ```
- [架构设计]<br>
  采用了类Intel X86指令集<br>
  借鉴了AT&T指令后缀，使用immu(i),memory(m),register(r)来表示指令参数<br>
  分为双后缀（例如movrr,movrm）和单后缀指令（例如addr,addi）<br>
  采用block块指令结构，指令集由指令块组成<br>
  支持部分高级语法，强类型（未实现）<br>
  不采用栈模型，全局64位寻址模式，有r0-r254的8字节寄存器和heap<br>
